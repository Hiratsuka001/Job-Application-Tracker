# Job Application Tracker

A streamlined web application designed to help job seekers organize and monitor their employment applications. This project provides a clean interface to track the status of various job listings, from initial discovery to interview stages or rejections.

---

## 🚀 Live Demo
https://hiratsuka001.github.io/Job-Application-Tracker/

## 📋 Project Overview
The **Job Application Tracker** is a functional dashboard that allows users to manage their job search workflow efficiently. It features real-time statistics, dynamic filtering, and status management for each application, ensuring you never lose track of where you stand with potential employers.

## ✨ Key Features
* **Application Management**: Mark jobs as "Interviewing" or "Rejected" to keep your list updated.
* **Real-time Statistics**: Automatically updates counts for total jobs, active interviews, and rejections.
* **Dynamic Filtering**: Filter your job list by "All", "Interview", or "Rejected" status to focus on specific applications.
* **Job Removal**: Delete job cards from your view with a single click.
* **Empty State Handling**: Displays a helpful "No jobs available" illustration and message when no applications match the current filter or all are deleted.
* **Responsive Design**: A fully responsive layout built with Tailwind CSS that works seamlessly on mobile and desktop devices.

## 🛠️ Technology Stack
* **HTML5**: Semantic structure.
* **CSS3 (Tailwind CSS & DaisyUI)**: For modern, responsive styling and pre-built UI components.
* **JavaScript (ES6+)**: Handles DOM manipulation, filtering logic, and state updates.
* **Google Fonts**: "Geist" font for a clean, professional look.
* **Font Awesome**: Icons for actions like deleting entries.

## 📦 Dependencies
This project uses the following CDN-based libraries:
* **Tailwind CSS**: Utility-first CSS framework.
* **DaisyUI (v5)**: Tailwind CSS components for buttons, stats, and cards.
* **Font Awesome (v7.0.1)**: Scalable vector icons.

## 💻 Local Setup Guide
To run this project on your local machine, follow these steps:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/job-application-tracker.git
    ```
2.  **Navigate to the Directory**:
    ```bash
    cd job-application-tracker
    ```
3.  **Open the Project**:
    Simply open the `index.html` file in any modern web browser.
    * *Alternatively*, use a development server like **Live Server** in VS Code for a better experience.
